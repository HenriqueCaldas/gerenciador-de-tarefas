# 🚀 API de Gerenciamento de Tarefas Avançado

Este projeto é um ecossistema completo de desenvolvimento de software, simulando todo o ciclo de vida de um produto real (SDLC). O objetivo é demonstrar práticas avançadas de engenharia de software, arquitetura, testes automatizados e cultura DevOps.

## 🛠️ Tecnologias e Ferramentas (Evolução do Projeto)
* **Linguagem Principal:** Java 21
* **Gerenciador de Dependências:** Maven
* **Testes Unitários:** JUnit 5

## 📌 Funcionalidades do Escopo (Simples e Direto)
* Criar uma tarefa (com título, descrição, data de vencimento e prioridade).
* Listar todas as tarefas.
* Marcar tarefa como concluída.
* Excluir uma tarefa.
